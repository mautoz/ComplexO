## Dicas

O Windows possui problemas para quem utiliza antivírus. Vocês podem até compilar, mas é capaz que não execute. Sendo assim, mantenham desativado durante os estudos!

## Compilação

```
$ gcc -o <nomeDoPrograma> <nomeDoPrograma.c>
```

## Execução

Para Windows: 
```
$ .\<nomeDoPrograma>
```
ou 
```
$ .\<nomeDoPrograma>.exe
```
Para Ubuntu: 
```
$ ./<nomeDoPrograma>
```
